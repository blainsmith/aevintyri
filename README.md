# Ævintýri

Text adventure game engine built with Hare.

## Running

```
> hare run cmd/main.ha
```

## Connecting

You can connect to the server with Telnet.

```
> telnet 0.0.0.0 7913
Trying 0.0.0.0...
Connected to 0.0.0.0
Escape character is '^]'.
+----------------------+
| Welcome to Ævintýri! |
+----------------------+

» █
```

## Testing

```
> hare test
```
